title: "lists"
---
##### Input

~~~text
1. First item
2. Second item
3. Third item

* First item
* Second item
* Third item
~~~

##### Output

~~~html
<ol>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ol>

<ul>
  <li>First item</li>
  <li>Second item</li>
  <li>Third item</li>
</ul>
~~~
